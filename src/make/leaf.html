Leaf.mk

Intro

Leaf.mk is a file that is used for all leaves in a directory tree. It defines a rule for converting files into POD format.

Rules

pod

make pod

Greps the files in that directory for a line ``DOPOD''. This is a trigger that the file needs to be converted to documentation. Then all lines that start with the comment: # POD get pulled out, the comment stripped off of them, and passed through the pod2html intepreter that is part of the perl distribution.

Tips for writing POD include:

  • Don't put anything on blank lines, they won't pass through the interpreter.

  • Do start your documentation on LINE 1 !

  • Use =head1 and =head2 commands

  • full pod docs are at http://www.perldoc.com/perl5.6.1/pod/perlpod.html|pod
